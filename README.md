# lzkj-data-web-pack

## 介绍

lzkj-data-web-pack 是一个基于 `TypeScript` 和 `React` 开发的前端工具库，为了解决陆泽科技数据业务部门前端开发中多个项目中重复开发的问题，提高开发效率，统一前端开发规范，提供了一些常用的React 组件和工具函数。

## 安装

```bash
npm i lzkj-data-web-pack --save
```
