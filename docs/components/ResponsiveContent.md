## API

| 属性 | 说明 | 类型 | 默认值 |
| --------------- | ----- | ---------------- | ------------------ |
| <div style="width: 120px">**width**</div> | <div style="width: 500px">容器宽度</div> | <div style="width: 200px">`number`</div> | <div style="width: 100px">`*`</div> |

| **style** | 容器样式 | `object` | `{}` |
| **children** | 容器的内容 | `React.ReactNode` | `-` |
